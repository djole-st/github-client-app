How to use Application

On home page, you will see small search form where you should enter Github Repository Url and GitHub Access Token.

After you perform search, list of GitHub issues for selected repository will appear.

Clicking on Issue Title, you will be redirected to issues details page, where you will see, basic information about selected issue, issues comments and reactions.

You data is NOT being saved anywhere!

Github Url Example: https://github.com/djole-st/github-client-app

Token for accessing my repositories is being provided in my application on your website.

For Accessing your own repositories, you will need to crete your own token and use it here in application.
